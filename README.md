# IMC-Calculator
<h1>O Que é IMC?</h1>
<p>Índice de Massa Corporal, é a quantidade de massa corporal que uma pessoa possui</p>
<h1>O que Faz?</h1>
<b>Este Programa simples ajuda a fazer o cálculo exato da sua massa corporal, com base nas suas especificações físicas</b>
<h1>Tech's Usadas </h1>
<p>HTML5</p>
<p>CSS3 (Flexbox)</p>
<p>Javascipt</p>
